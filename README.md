# universidade-corporativa-frontend

## Requirements

- Node.js (& npm)
- Yarn

## Install

```sh
  $ yarn install
```

## Serve

```sh
  $ yarn serve --dev --open
```

## Linter

```sh
  $ yarn lint --fix
```

## Build

```sh
  $ rm -rf dist && ./build.sh  # ou
  $ rm -rf dist && NODE_ENV=production yarn run build
```
